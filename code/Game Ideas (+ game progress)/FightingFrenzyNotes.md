Game Concept: 1v1 Arena Sword Fighter
Overview

A fast-paced 1v1 2D PvP arena combat game where players engage in sword-based duels across randomly generated maps filled with obstacles, platforms, and power-enhancing objects. The game blends technical melee combat with chaotic map dynamics and a single, high-stakes weapon: the Golden Gun.
